# Математический тренажёр для 5 класса

Интерактивный тренажёр для отработки математических навыков учащихся 5 класса.

## Описание проекта

Проект создан для самостоятельной тренировки и повторения основных тем по математике.  
Тренажёр помогает ученикам решать задания, сразу проверять ответы и видеть результат.

## Возможности тренажёра

- задания по разным темам математики;
- автоматическая проверка ответов;
- подсчёт правильных ответов;
- отображение результата;
- удобный интерфейс для школьников;
- работа прямо в браузере без установки дополнительных программ.

## Для кого предназначен

Тренажёр подходит для:

- учащихся 5 класса;
- самостоятельной подготовки дома;
- повторения материала на уроке;
- дополнительных занятий;
- индивидуальной работы с учеником.

## Как открыть тренажёр

1. Откройте файл `index.html` в браузере.
2. Выберите тему.
3. Выполните задания.
4. Проверьте результат.
